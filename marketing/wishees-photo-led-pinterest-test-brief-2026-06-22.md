# Wishees Photo-Led Pinterest Test Brief - 2026-06-22

Purpose: test whether realistic, high-quality lifestyle photos with short text overlays perform better than the current graphic-style Wishees pins.

## Why Test This

Pinterest is visual search. The current assets are clean and on-brand, but they read more like SaaS graphics. For gift-intent searches, a realistic birthday cake, baby shower table, wrapped gifts, or wedding gift table should feel more native to Pinterest and may earn stronger saves and clicks.

Use photo-led pins as an A/B test against the existing graphic-style pins. Do not replace all assets until there is signal.

## Creative Rules

1. Use 1000x1500, 2:3 vertical images.
2. Use realistic photo backgrounds, not abstract illustrations.
3. Keep overlay text to 10 words or fewer.
4. Make the use case obvious without reading the description.
5. Keep `Wishees.com` small but visible.
6. Avoid fake app screenshots unless the image is intentionally product-led.
7. Avoid cluttered text blocks; the photo should carry the emotion, the text should clarify the use case.
8. Keep one use case per pin: birthday, baby shower, wedding, grandparents, class gift, creator gift guide.

## Recommended Posting Test

Do not publish all photo-led variants at once.

Best sequence:

1. Finish/publish current staged Pin 3 or publish Photo Pin B instead if we decide the photo-led test should start immediately.
2. Wait at least 24 hours.
3. Compare early Pinterest stats:
   - impressions/views
   - saves
   - outbound clicks
   - comments
4. If Photo Pin B performs better than the graphic baby-shower pin, make the next Pinterest batch photo-led.

## Decision Rule

Use photo-led assets as the new default if one of the first three variants gets:

- more saves than the graphic-style pin for the same angle, or
- any outbound click when the graphic-style pin has none, or
- a comment/question that shows the use case is understood.

Keep graphic-style assets only for creator/affiliate education posts where the idea is harder to show through a photo.
