# Wishees Post-Pin-3 Measurement Playbook - 2026-06-22

Use this after Pinterest Pin 3, baby shower wishlist, is published. This playbook does not authorize new outreach.

## Purpose

Pin 3 tests whether practical baby-shower gift intent can create a stronger discovery signal than the first broad birthday/wedding posts.

Primary learning:

Does a concrete parent/newborn use case create saves, clicks, comments, mailbox replies, affiliate requests, or product-click movement?

## Immediately After Publish

Record in `wishees-posting-tracker-2026-06-22.csv`:

1. `Pin 3` status: `Published`.
2. `Published_URL`: live Pinterest URL.
3. `Published_Date`: publish date.
4. `Notes`: any Pinterest warning, rejection, missing board, or language/link issue.

Then check:

1. Open the published pin URL.
2. Confirm the title is `Baby shower wishlist made easy`.
3. Confirm the destination link is `https://www.wishees.com/?lang=en`.
4. Confirm the visual is the baby-shower asset, not the wedding or birthday asset.
5. Confirm the pin is not accidentally saved to the wrong board.

If any of these fail, fix the pin if Pinterest allows editing. If editing is blocked, record the exact issue in the tracker and do not create a duplicate unless the link or visual is wrong.

## 15-Minute Check

Check only for publishing quality:

- Pin URL opens.
- Destination link opens.
- No wrong-language landing page.
- No Pinterest warning.
- No obvious duplicate pin draft left unpublished.

Record only if something is broken.

Do not judge performance after 15 minutes.

## 24-Hour Check

Update `wishees-posting-tracker-2026-06-22.csv`:

- Views or impressions if Pinterest exposes them.
- Saves if visible.
- Outbound clicks if visible.
- Comments if visible.

Also check:

- `growth@wishees.com` mailbox for creator replies or bounces.
- Affiliate requests page for any new request after Pin 3 publish time.
- Product clicks page for any new click after Pin 3 publish time.
- Growth statistics for share-link opens or connections.

Add one row to `wishees-growth-measurement-ledger-2026-06-22.csv`.

Suggested ledger trigger:

`pin3_24h_check`

## 72-Hour Check

Use the same checks as 24 hours, but make a decision:

### Continue Pinterest baby-shower angle if:

- Pin 3 gets saves, comments, or outbound clicks, even without affiliate signup.
- Product clicks or share-link opens increase after the pin.
- A creator reply mentions baby shower, parenting, newborn gifts, family gifts, or registry ideas.

Next action:

Publish a second parent/newborn-adjacent pin or use baby-shower language in the next outreach batch.

### Continue broad channel warming if:

- Pin 3 has reach but no clicks/comments.
- TikTok or YouTube still has views but no conversion.
- No mailbox replies yet.

Next action:

Publish YouTube Short 2, wedding registry supplement, because it tests the "works alongside registry" positioning.

### Hold outreach if:

- No replies.
- No affiliate requests.
- No product-click movement.
- No social comments or saves.

Next action:

Keep batch 2 paused until the 2026-06-27 follow-up decision point.

### Adjust copy if:

- People click but do not connect/register.
- Comments or replies show confusion about whether Wishees replaces Babylist/Zola/Amazon registries.
- The landing page appears in German or unclear language for US visitors.

Next action:

Tighten copy around "wishlist supplement" and "works alongside existing tools."

## Decision Thresholds

Do not scale outreach from views alone.

Minimum signal to justify more outreach:

- One positive creator reply, or
- one affiliate request, or
- one saved affiliate tag, or
- one meaningful comment/question, or
- clear new product-click movement after publish time.

Minimum signal to justify another channel post:

- Published URL is healthy and no platform warning exists.

## Exact Next Action If Pin 3 Performs

If Pin 3 creates any meaningful signal, prepare a baby-shower-specific outreach angle for the next clean leads:

- Pragmatic Mom
- Diamond Tales
- Pjs and Paint follow-up if no reply by 2026-06-27

Message angle:

`Wishees can be a lightweight baby-shower wishlist supplement for readers who already use Amazon, Target, Babylist, or registry tools. It helps guests share and organize gift ideas without asking creators or parents to switch systems.`

## Exact Next Action If Pin 3 Does Not Perform

Publish YouTube Short 2, wedding registry supplement, before sending more outreach.

Reason:

It tests the more strategic positioning: Wishees as a supplement to existing registries, not a replacement.
