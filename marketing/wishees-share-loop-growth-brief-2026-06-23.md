# Wishees Share Loop Growth Brief - 2026-06-23

Source: growth dashboard interpretation from side chat.

## Diagnosis

Wishees is getting onboarding activity, but the share loop is weak.

Current important numbers:

- 21 homepage wish starts
- 12 signups completed
- 8 first wishes saved
- 5 share links created
- 14 share links opened
- 2 connections created
- current week: 0 shares, 1 link open, 0 connections, 0 gift reservations

The core bottleneck is not basic wish creation. The bottleneck is:

> Users create or start wishes, but too few create and share links.

## Growth Rule

For now, every Facebook post should push one action:

> Create one birthday/family wishlist and share the link.

Do not split the CTA into affiliate, marketplace, product browsing, or broad feature education.

## Product Activation Direction

After signup or first wish, Wishees should guide the user through:

1. Add 3 wishes.
2. Create share link.
3. Copy/share to family or friends.

The product should make this feel like the natural next step, not an optional extra.

## Occasion Templates

Add or emphasize templates for:

1. Birthday wishlist
2. Baby shower wishlist
3. Wedding extra gift list
4. Family gift list
5. Kids gift list

Templates should reduce blank-page friction and lead directly into sharing.

## Facebook Measurement

Facebook success is not views.

Measure:

1. Did Facebook create new wishes?
2. Did it create share links?
3. Did those links open?
4. Did any connection or gift reservation happen?

If Facebook views rise but share links stay at 0, the next fix is onboarding/share CTA, not more outreach.

## Outreach Decision

Do not scale outreach yet.

Outreach can wait until the page and product loop prove that visitors can move from:

Facebook post -> create wishlist -> share link -> link open -> connection/reservation.

## Next Experiment

Publish the Facebook intro/birthday post, then watch the dashboard for 24 hours:

- shares created
- links opened
- connections
- gift reservations

If shares remain 0, fix onboarding/share CTA before sending more outreach.
