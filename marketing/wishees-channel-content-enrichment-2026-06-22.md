# Wishees Channel Content Enrichment - 2026-06-22

Purpose: make each Pinterest, YouTube, and TikTok post more discoverable, more useful to the right visitor, and easier to measure while first-batch outreach waits for replies.

Default visual style:

Use `wishees-photo-video-creative-style-guide-2026-06-22.md`. Future Wishees photos and videos should use the realistic photo-led style unless the content specifically needs product UI or affiliate-mechanics explanation.

Core message:

Wishees is a simple shareable wishlist layer for gift moments. It works alongside Amazon, registries, creator storefronts, and affiliate tools. It does not ask creators or families to replace what already works.

## Channel Rules

1. Pinterest should target search intent.
   - Use phrases people actually search when planning gifts.
   - Make the image explain the use case before the description is read.
   - For concrete family/gift moments, prefer realistic lifestyle photos with short overlay text over SaaS-style graphics.
   - Link broad gift-moment posts to `https://www.wishees.com/?lang=en`.
   - Link creator/affiliate posts to `https://www.wishees.com/affiliate/program?lang=en`.

2. YouTube Shorts should build credibility.
   - Keep titles specific to one gift moment.
   - Use descriptions that repeat the "works alongside" idea.
   - Do not over-explain affiliate mechanics in broad family posts.

3. TikTok should test simple hooks.
   - Use no-hashtag captions until duplicate hashtag behavior is confirmed fixed.
   - Use plain, human lines that sound like a real gift problem.
   - Do not publish the next TikTok until the public caption is checked immediately after posting.

4. Every post should test one learning.
   - If the post gets views but no clicks, improve CTA and landing-page match.
   - If the post gets saves but no clicks, create more Pinterest assets in that angle.
   - If the post gets comments/questions, use that language in the next outreach batch.
   - If creator-facing content performs better than family-gift content, shift the next email batch toward creators with existing gift guides.

5. Photo-led Pinterest assets should be tested.
   - Use realistic 1000x1500 images for birthday, baby shower, wedding, grandparents, and class-gift searches.
   - Keep overlay text to 10 words or fewer.
   - Keep `Wishees.com` small but visible.
   - Use graphic-style assets mainly for creator/affiliate education where a photo alone cannot explain the product.
