# Wishees Chrome Publishing Runbook

Use this in the logged-in Chrome sessions for Pinterest, YouTube, and TikTok.

Goal: make the social profiles credible enough to support the first affiliate outreach batch. This is not the main growth channel yet; Pinterest discovery -> blog email/form remains the active channel.

## Before Publishing

1. Open `wishees-profile-readiness-checklist-2026-06-22.md`.
2. Confirm each profile has name, avatar/logo, bio, and link where possible.
3. Do not add copy that says Wishees pays creators.
4. Do not position Wishees as replacing Amazon, LTK, ShopMy, Zola, Babylist, Giftster, Elfster, or MyRegistry.
5. After publishing each item, paste the live URL into `wishees-posting-tracker-2026-06-22.csv`.
6. For US campaign links, keep `lang=en` in every Wishees URL.

Note: unauthenticated shared-wish URLs show preview items, then ask visitors to sign in or register before the full wishlist is connected. For creator conversion, prefer the affiliate-program link; use shared wish links as examples of gift-list moments.

## After Publishing

For each item:

1. Paste the published URL into `wishees-posting-tracker-2026-06-22.csv`.
2. Add published date.
3. Leave clicks/views/comments blank until the first measurement pass.
4. If the platform rejects an upload, record the error in the Notes column.
5. Do not start outreach until the first 5 outreach messages are approved.
