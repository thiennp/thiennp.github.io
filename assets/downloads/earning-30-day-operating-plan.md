# 30-Day Remote Earning Operating Plan

This plan focuses on no-upfront-payment earning systems. It avoids card-required offers, deposits, crypto, fake reviews, and automated platform abuse.

## Week 1: Cashflow setup

**Goal:** Complete legitimate profiles and prepare for approval.

1. Complete UserTesting practice test personally.
2. Create TestingTime profile.
3. Create Testbirds or uTest profile.
4. Prepare a device inventory list: laptop, phone, tablet, OS versions, browsers.
5. Save payout method notes in the tracker.
6. Practice one five-minute think-aloud test.

**Output:** Three active profiles, one practice recording, tracker updated.

## Week 2: AI evaluator pipeline

**Goal:** Apply to higher-value evaluation platforms.

1. Apply to DataAnnotation.
2. Apply to TELUS Digital AI Community roles that match your country/language.
3. Apply to Welocalize evaluator roles.
4. Practice rating explanations using the AI evaluator kit.
5. Record which assessments are pending.

**Output:** Three applications, assessment notes, next follow-up dates.

## Week 3: Digital product shelf

**Goal:** Turn useful internal assets into public downloads.

1. Publish the user testing practice kit.
2. Publish the bug report template pack.
3. Publish the AI evaluator practice kit.
4. Create a simple product description for each.
5. Decide which marketplace to use for paid versions.

**Output:** Three free lead magnets and one paid-product plan.

## Week 4: Affiliate and lead generation

**Goal:** Build pages that can earn later without inventory.

1. Write a "best user testing platforms" comparison page.
2. Write a "best AI evaluator platforms" comparison page.
3. Add disclosure text before any affiliate links.
4. Add contact call-to-action for QA review or website feedback services.
5. Review analytics only after pages have useful content.

**Output:** Two comparison pages and one service lead path.

## Weekly review

Every Sunday, update:

- Applications submitted
- Assessments completed
- Tasks accepted
- Payouts received
- Hours spent
- Hourly rate
- Platforms to drop
- Next best action
