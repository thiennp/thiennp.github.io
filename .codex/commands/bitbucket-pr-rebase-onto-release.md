# Bitbucket PR: rebase onto release

Integrate a Bitbucket PR with **`origin/release`**: fetch the source branch, **rebase**, resolve conflicts, run **`npm run pre-deploy`** in **`frontend/`**, and commit any resolutions or artifacts the gate requires.
