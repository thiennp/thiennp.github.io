# Git push current branch

Push the **currently checked-out** branch to **origin** and set upstream tracking.

Confirm **`git push`** with the user when they have not already asked to push (**`.cursor/rules/commit-workflow.mdc`**).

From **repository root**, use your Git integration to push the active branch to **origin** with upstream set (same intent as **`git push -u origin <current-branch>`**).

Orientation: **`.cursor/rules/commit-workflow.mdc`**, **`.cursor/rules/git-workflow.mdc`**.
